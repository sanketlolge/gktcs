About the Course

The jQuery UI training course is designed to provide knowledge and skills to become a successful UI Developer. It starts with the fundamental concepts of jQuery DOM Traversing, jQuery Events API, jQuery Effects to advance topics of jQuery UI Widgets, jQuery Mobile etc. Participants will also get to implement one realtime UI Project to a web application and mobile app.

Course Objectives

After the completion of 'jQuery UI Training' at GKTCS, you will be able to: 
1. Understand Basic & Advanced methods of jQuery Library
2. Apply various jQuery methods in building UI projects
3. Develop Rich Interactive Web pages
4. Learn where to use and how to use jQuery UI Widgets, Events jQuery Mobile
5. Apply jQuery Library to create Highly Rich Web & Mobile Apps
6. Work on a real time UI for Web and Mobile App.

Who should go for this Course?

The course is designed for professionals who want to learn jQuery UI & Mobile Library and wish to apply it on Web & Mobile Front end. The following professionals can go for this course:
1. Front End UI Developers who want develop Web Apps 
2. Professionals who want to build UI widgets for Enterprise Web Apps
3. Professionals who want to build HTML5 based Mobile Apps
4. UI Designers & Developers 
5. Freelance Web designers/developers

What are the pre-requisites for this Course?

The pre-requisite for this course include knowledge of Basic HTML, CSS & Javascript

1. Introducing jQuery

Learning Objectives - In this module, you will understand about jQuery and overview of what jQuery does for web pages. you will also learn Basics of HTML & CSS and how jQuery is used to manipulate your DOM & CSS Selectors. 

Topics - Intro to jQuery & History, The jQuery Object($), Basic DOM HTML Document/Page Types, CSS Selectors, IDE's for jQuery, jQuery CDN & Installation. jQuery Functions, Selectors, Querying DOM, Traversing the DOM. jQuery Selectors, jQuery Basic Functions. DOM Traversing.

2. DOM Manipulation & jQuery Effects

Learning Objectives - In this module, you will learn how jQuery is used manipulate DOM easily, Also you will learn about readily available jQuery effects that can be applied to web page to make Rich Interactive Websites. 

Topics - Adding and Removing DOM elements, Iterating with each(), DOM elements, Modifying Properties and Attributes. jQuery Filters, jQuery Effects, hide/show, fading methods, chaining, Animate Function.

3. jQuery Forms & Events

Learning Objectives - In this module, you will learn jQuery Event event delegation and various events type in 

Topics - jQuery Form Selectors, Form Validation, Form Submission, Events, Event Handling, jQuery on() & off() , Custom Events and Triggers.

4. jQuery AJAX & Utilities

Learning Objectives - In this module, you will learn about AJAX and how you can communicate with server using jQuery inbuilt functions and other jQuery utilities. 

Topics - AJAX using jQuery, Interacting with Server-Side, PHP and $.each and Templating, Twitter/Github API, Deferreds.

5. jQuery Plugins

Learning Objectives - In this module, you will learn about jQuery plugins, how to use them into your web projects and basic plugin development. 

Topics - jQuery 3rd party plugins, Plugin Implementation, Plugin Development.

6. jQuery UI

Learning Objectives - In this module, you will learn about jQuery UI and single UI Component. We will look through the APIs of each Component. 

Topics - Accordion Widget, Tabs Widget, Tooltip Widget, Menu Widget, Widget, Datepicker Widget, jQuery Mobile.

7. jQuery Mobile Introduction

Learning Objectives - In this module, you will learn about Basics of jQuery Mobile. And also various UI components available in jQuery Mobile. 

Topics - jQuery Mobile CSS Framework, jQuery Mobile Events, Mobile Widgets

8. Project
